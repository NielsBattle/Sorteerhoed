### Installatie guide
1. Zorg dat je python 3.8 geïnstallerd hebt.
2. Zorg dat je pygame en pandas geïnstalleerd hebt.
3. Clone de repository naar je computer.
4. voer de app.py uit volgens ```python app.py```.
